tkDND necessary libraries are contained in this folder for linux.
If for some reason does not work you can attempt to download
and install using the following instructions:

To activate drag and drop.

Go to tkDND homepage
http://sourceforge.net/projects/tkdnd/

Donwload version binary 2.6 for your system

Or alternatively get the sources and compile it.
Download sources version 2.6
http://sourceforge.net/projects/tkdnd/files/TkDND/TkDND%202.6/

Unpack downloaded file somewhere in your filesystem
Place the contents inside the tkdnd folder

Launch Artscript, it should accept droped files from the filemanager
